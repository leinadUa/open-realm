# Golden-image render manifest for tools/parity/render_golden.sh
#
# One render per line:  name | mpq | model | extra mdxtool args
#   name   = output basename (golden/<name>.png)
#   mpq    = archive (relative to the data dir) that contains the model AND its textures
#   model  = MPQ path to the .mdx (use backslashes)
#   extra  = extra mdxtool flags (camera, anim). --frame/--seed default deterministically.
#
# Pick models that are self-contained in one archive and exercise distinct
# renderer paths (flipbook water, team color, particles, alpha UI, geometry).
